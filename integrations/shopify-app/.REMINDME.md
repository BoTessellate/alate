# Shopify App Setup Instructions

## Prerequisites

1. **Shopify Partner Account**
   - Sign up at [partners.shopify.com](https://partners.shopify.com)
   - Create a new app in the Partner Dashboard

2. **Development Store**
   - Create a development store for testing
   - Install the app on your dev store

## Environment Variables

Add these to your `.env` file:

```env
# Shopify App Credentials (from Partner Dashboard)
SHOPIFY_API_KEY=your_api_key_here
SHOPIFY_SECRET=your_api_secret_here

# App Configuration
APP_URL=https://your-app-domain.com
DEV_STORE_URL=your-store.myshopify.com
```

## Setup Steps

### 1. Create App in Partner Dashboard

1. Go to [Shopify Partners](https://partners.shopify.com)
2. Navigate to Apps → Create app
3. Choose "Create app manually"
4. Set app name: "Mood Layer"
5. Copy the API key and secret

### 2. Configure App URLs

In the Partner Dashboard, set:

- **App URL**: `https://your-domain.com`
- **Allowed redirection URLs**:
  - `https://your-domain.com/auth/callback`
  - `https://your-domain.com/api/auth/callback`

### 3. Configure Webhooks

In Partner Dashboard → Webhooks:

1. Add webhook for `products/create` → `https://your-domain.com/webhooks/products`
2. Add webhook for `products/update` → `https://your-domain.com/webhooks/products`
3. Add webhook for `products/delete` → `https://your-domain.com/webhooks/products`
4. Add webhook for `app/uninstalled` → `https://your-domain.com/webhooks/app-uninstalled`

### 4. Required Scopes

The app requests these scopes:
- `read_products` - Read product data
- `write_products` - Update product data (optional)
- `read_inventory` - Read inventory levels

### 5. Install on Dev Store

1. Go to your development store admin
2. Apps → Install app
3. Authorize the requested scopes
4. Complete OAuth flow

## Testing

### Test OAuth Flow
```bash
curl "http://localhost:3000/auth?shop=your-store.myshopify.com"
```

### Test Webhook (local)
```bash
curl -X POST http://localhost:3000/webhooks/products \
  -H "Content-Type: application/json" \
  -H "X-Shopify-Topic: products/create" \
  -H "X-Shopify-Shop-Domain: your-store.myshopify.com" \
  -d '{"id": 123, "title": "Test Product"}'
```

## Troubleshooting

### "Invalid HMAC" errors
- Ensure SHOPIFY_SECRET matches Partner Dashboard
- Verify raw body is being used for HMAC calculation

### OAuth redirect errors
- Check allowed redirect URLs in Partner Dashboard
- Ensure APP_URL matches exactly

### Webhook not receiving
- Verify webhook URLs are publicly accessible
- Check Shopify Partner Dashboard → Webhooks for delivery status
